DisArray

DisArray is a chat messaging app inspired by Discord. DisArray allows users to create servers, channels within servers, and chat with friends. Features include: secure authentication for user login and sign up, creation and deletion of servers, creation, deletion, and editing within servers, invite codes for channels, user uploads with Amazon Web Services, and of course, chatting! Chat rooms broadcasts to all members part of a channel that updates in real time using Action Cable.

The backend technologies: Ruby on Rails-Action Cable, PostgresSQL, and Amazon Web Services. Hosting: Heroku. On the frontend: React/Redux, REACT DOM, React Router, jQuery, and webpack were used.

Possible future additions: direct messaging capabilities between users
