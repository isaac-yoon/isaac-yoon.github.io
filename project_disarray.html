DisArray

Inspired by DISCORD, DisArray or "this array" is a chatting app.


Features include: secure authentication for user login and sign up, creation and deletion of servers and channels within servers, custom codes for channel invites, ability to upload server background images using Amazon Web Services (AWS), and of course, chatting! Chat rooms broadcasts to all members part of a channel that updates in real time using Ruby's Action Cable.


BACK-END: Ruby on Rails-Action Cable, PostgresSQL, and Amazon Web Services (AWS).

Hosting: Heroku

FRONT-END: React/Redux, REACT DOM, React Router, jQuery, and Webpack.

Possible future additions: direct messaging capabilities between users
